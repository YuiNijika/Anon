CLI 命令行系统 ​

CLI（Command Line Interface）命令行系统允许开发者通过终端运行自定义命令，执行批处理任务、维护操作等。

快速开始 ​

注册第一个命令 ​

php
Anon_System_Console::command('hello', function($args) {
    Anon_System_Console::info('Hello, World!');
    return 0;
}, '输出问候语');

运行命令 ​

bash
php index.php hello

命令注册 ​

基本用法 ​

php
Anon_System_Console::command(
    'command:name',           // 命令名
    $handler,                 // 处理器
    '命令描述'                // 描述（可选）
);

处理器类型 ​

1. 闭包函数 ​

php
Anon_System_Console::command('cache:clear', function($args) {
    Anon_System_Console::info('清除缓存...');
    
    // 清除缓存逻辑
    Anon_Cache::clear();
    
    Anon_System_Console::success('缓存已清除');
    return 0;
}, '清除所有缓存');

2. 类方法 ​

php
class CacheClearCommand
{
    public function handle($args)
    {
        Anon_System_Console::info('清除缓存...');
        Anon_Cache::clear();
        Anon_System_Console::success('缓存已清除');
        return 0;
    }
}

Anon_System_Console::command('cache:clear', CacheClearCommand::class, '清除缓存');

3. 可调用类 ​

php
class SendEmailCommand
{
    public function __invoke($args)
    {
        // 发送邮件逻辑
        return 0;
    }
}

Anon_System_Console::command('mail:send', SendEmailCommand::class, '发送邮件');

输出方法 ​

info() - 普通信息 ​

php
Anon_System_Console::info('这是一条普通信息');
// 输出：这是一条普通信息

php

// 输出：绿色的"操作成功完成"

php

// 输出：红色的"发生错误"

warning() - 警告信息（黄色） ​

php
Anon_System_Console::warning('这是一个警告');
// 输出：黄色的"这是一个警告"

line() - 自定义输出 ​

php
Anon_System_Console::line('自定义输出内容');
Anon_System_Console::line(''); // 空行

完整示例 ​

示例 1：数据库备份命令 ​

php
Anon_System_Console::command('db:backup', function($args) {
    Anon_System_Console::info('开始备份数据库...');
    
    try {
        $db = Anon_Database::getInstance();
        $config = $db->getConfig();
        
        $backupDir = ANON_ROOT . 'backups/';
        if (!is_dir($backupDir)) {
            mkdir($backupDir, 0755, true);
        }
        
        $filename = 'backup_' . date('Y-m-d_H-i-s') . '.sql';
        $filepath = $backupDir . $filename;
        
        // 执行 mysqldump
        $command = sprintf(
            'mysqldump -h%s -u%s -p%s %s > %s',
            $config['host'],
            $config['username'],
            $config['password'],
            $config['database'],
            escapeshellarg($filepath)
        );
        
        exec($command, $output, $returnCode);
        
        if ($returnCode === 0) {
            $size = round(filesize($filepath) / 1024 / 1024, 2);
            Anon_System_Console::success("数据库备份成功：{$filename} ({$size}MB)");
            return 0;
        } else {
            Anon_System_Console::error('数据库备份失败');
            return 1;
        }
    } catch (Exception $e) {
        Anon_System_Console::error('备份失败：' . $e->getMessage());
        return 1;
    }
}, '备份数据库到 backups 目录');

示例 2：清理日志命令 ​

php
Anon_System_Console::command('logs:clean', function($args) {
    $days = isset($args[0]) ? (int)$args[0] : 30;
    
    Anon_System_Console::info("清理 {$days} 天前的日志...");
    
    $logDir = ANON_ROOT . 'logs/';
    if (!is_dir($logDir)) {
        Anon_System_Console::warning('日志目录不存在');
        return 0;
    }
    
    $count = 0;
    $files = scandir($logDir);
    
    foreach ($files as $file) {
        if ($file === '.' || $file === '..') {
            continue;
        }
        
        $filepath = $logDir . $file;
        if (!is_file($filepath)) {
            continue;
        }
        
        $fileTime = filemtime($filepath);
        $daysOld = (time() - $fileTime) / 86400;
        
        if ($daysOld > $days) {
            unlink($filepath);
            $count++;
            Anon_System_Console::line("  已删除：{$file}");
        }
    }
    
    Anon_System_Console::success("清理完成，共删除 {$count} 个文件");
    return 0;
}, '清理旧日志文件 [days=30]');

示例 4：生成站点地图命令 ​

php
Anon_System_Console::command('sitemap:generate', function($args) {
    Anon_System_Console::info('生成站点地图...');
    
    $db = Anon_Database::getInstance();
    
    // 获取所有文章
    $posts = $db->db('posts')
        ->where('status', '=', 'published')
        ->all();
    
    $xml = new DOMDocument('1.0', 'UTF-8');
    $urlset = $xml->appendChild($xml->createElement('urlset'));
    $urlset->setAttribute('xmlns', 'http://www.sitemaps.org/schemas/sitemap/0.9');
    
    // 添加首页
    $url = $urlset->appendChild($xml->createElement('url'));
    $loc = $url->appendChild($xml->createElement('loc'));
    $loc->nodeValue = Anon_System_Env::get('app.url', 'https://example.com');
    
    // 添加文章页
    foreach ($posts as $post) {
        $url = $urlset->appendChild($xml->createElement('url'));
        $loc = $url->appendChild($xml->createElement('loc'));
        $loc->nodeValue = Anon_System_Env::get('app.url') . '/post/' . $post['id'];
        
        $lastmod = $url->appendChild($xml->createElement('lastmod'));
        $lastmod->nodeValue = date('Y-m-d', strtotime($post['updated_at']));
    }
    
    $xml->formatOutput = true;
    $filepath = ANON_ROOT . 'sitemap.xml';
    $xml->save($filepath);
    
    Anon_System_Console::success("站点地图已生成：{$filepath}");
    return 0;
}, '生成 XML 站点地图');

别名注册 ​

为常用命令注册简短的别名：

php
Anon_System_Console::command('database:backup', $handler, '备份数据库');
Anon_System_Console::alias('db:backup', 'database:backup');

Anon_System_Console::command('cache:clear', $handler, '清除缓存');
Anon_System_Console::alias('cc', 'cache:clear');

使用：

bash
php index.php db:backup
php index.php cc

退出码 ​

退出码含义
0成功执行
1一般错误
2配置错误
3+自定义错误

使用示例 ​

php
Anon_System_Console::command('critical-task', function($args) {
    try {
        // 关键任务
        return 0; // 成功
    } catch (ConfigException $e) {
        Anon_System_Console::error('配置错误：' . $e->getMessage());
        return 2;
    } catch (Exception $e) {
        Anon_System_Console::error('执行错误：' . $e->getMessage());
        return 1;
    }
});

php
// ✅ 正确
'db:backup'

// ❌ 错误

'clearcache'

2. 错误处理 ​

始终捕获并报告错误：

php
Anon_System_Console::command('important-task', function($args) {
    try {
        // 业务逻辑
        Anon_System_Console::success('任务完成');
        return 0;
    } catch (Throwable $e) {
        Anon_System_Console::error('任务失败：' . $e->getMessage());
        Anon_Debug::error('CLI command failed', ['error' => $e->getMessage()]);
        return 1;
    }
});
