# API 端点

一句话：系统提供的所有API端点列表。

## 系统端点

- `GET /anon/debug/console` - Web调试控制台

## 认证端点

- `POST /auth/login` - 用户登录
- `POST /auth/logout` - 用户注销
- `GET /auth/check-login` - 检查登录状态
- `GET /auth/token` - 获取Token
- `GET /auth/captcha` - 获取验证码

## 用户端点

- `GET /user/info` - 获取用户信息

## 测试端点

- `GET /test/widget` - Widget测试接口
